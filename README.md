# Facebook AD Campaign Analysis
## The purpose of this project is to analyse our given dataset and whether there is a correlation between any of the categories and the conversion rate.

## We will also explore which gender's or age group's show more interest for particular ad campaigns so that a business could change there marketing method according to these results.
